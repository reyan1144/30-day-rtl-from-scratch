# Day 04 – Flip-Flops and Barrel Shifter

## Overview
This day focuses on sequential logic using flip-flops and combinational shifting using a barrel shifter.

---

## Topics Covered

- D Flip-Flop (DFF)
- Synchronous reset vs Asynchronous reset
- Enable logic in sequential circuits
- Barrel shifter concept

---

## Modules Implemented

### 1. D Flip-Flop (DFF)
- Synchronous reset  
- Asynchronous reset  
- Enable-controlled DFF  

### 2. Barrel Shifter
- 4-bit left barrel shifter  
- Variable shift using control signals  

---

## Key Concepts

- Flip-flops store data based on clock edges  
- Synchronous reset acts only on clock edge  
- Asynchronous reset acts immediately  
- Enable controls when data is updated  
- Barrel shifter performs shifting in a single step  

---

## What This Day Demonstrates

- Designing sequential circuits using clock  
- Understanding reset behavior  
- Implementing control logic (enable)  
- Building combinational shifting circuits  
- Verifying designs using testbenches and waveforms
