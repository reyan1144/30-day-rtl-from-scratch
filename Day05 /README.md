# Day 05 – Registers and Storage Elements

## Overview
This day focuses on multi-bit storage elements in digital design, building on the concept of D flip-flops.

---

## Topics Covered

- Register (multi-bit storage)
- Shift register (data shifting)
- Register file (multiple register storage)
- Difference between single-bit and multi-bit storage
- Sequential data movement

---

## Modules Implemented

### 1. Register
- 4-bit register  
- Synchronous reset  
- Enable-controlled data loading  

### 2. Shift Register
- 4-bit left shift register  
- Serial input  
- Enable-controlled shifting  

### 3. Register File
- 4 registers (4-bit each)  
- Write and read operations  
- Address-based access  

---

## Key Concepts

- Registers are built using multiple D flip-flops  
- Enable controls when data is updated  
- Shift registers move data across bits  
- Register files store multiple values using address selection  

---

## What This Day Demonstrates

- Designing multi-bit storage systems  
- Understanding data movement in sequential circuits  
- Using arrays for register storage  
- Implementing read/write operations
