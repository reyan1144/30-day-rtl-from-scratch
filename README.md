# 30-day-rtl-from-scratch

A structured journey to learn RTL design from scratch using Verilog/SystemVerilog, focusing on hands-on implementation, verification, and waveform analysis.

---

## Highlights

- RTL design from fundamentals to intermediate concepts
- Structured and consistent learning approach
- Code + Testbench + Waveform-based verification
- Focus on debugging and real understanding

---

## Approach

This repository focuses on learning RTL design through:
- Writing clean and synthesizable Verilog
- Verifying functionality using testbenches
- Analyzing behavior using waveforms
- Debugging real issues instead of just writing code

---

## Structure

Each day contains:
- Design files
- Testbench
- Waveform
- Explanation
